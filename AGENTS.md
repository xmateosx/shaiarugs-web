# Shaia Rugs

shaiarugs.com — fine antique oriental rugs, Williamsburg, Virginia, est. 1973.
Next.js 16 site with an admin section backed by Vercel Blob + Vercel KV.

Design principle: site motion stays understated. Hover zoom is a subtle 1.25x tease;
the lightbox (1.75x, pinch to 4x) is the inspection tool.
